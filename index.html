SKEWBThe optimal solutions to all Skewb positions stored in a 200MB txt file. The last column is used to show the solution moves required, with the rest of the columns conveying the state of the Skewb. (Except for the second last column, which is the inverse of the solution which represents what moves is needed to get the Skewb to that position)
SKEWBThe compressed version of the above file [37.8 MB]. All states are stored as 64 bit unsigned integers and solutions as 32 bit unsigned integers. Designed to be small enough to be able to fit on a VEX Robot brain [for a future project]
